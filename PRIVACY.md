# Privacy Policy

Spectra does not collect, transmit, or store your source code, configuration files, or cryptographic assets.

## The CLI
The Spectra CLI runs entirely locally. It does not phone home. It does not require an account. It does not send analytics. 

## The Playground (Web)
If you use the browser-based WASM playground, execution happens in your browser. 
If you use the Server mode in the playground, your code is processed in memory and immediately discarded. It is not written to disk or retained in logs.
