# Security Policy

Spectra is designed as a precision instrument for cryptographic intelligence. 
We take the security of this tool as seriously as the vulnerabilities it finds.

## Supported Versions
Only the latest major version is supported for security updates.

## No Telemetry Guarantee
Spectra explicitly does not include telemetry, tracking, or call-home functionality. 
Your codebase never leaves your machine during a local scan.

## Reporting a Vulnerability
If you find a security issue in Spectra, please email harshal@spectra.tools. 
Do not open a public issue. We will respond within 48 hours.
